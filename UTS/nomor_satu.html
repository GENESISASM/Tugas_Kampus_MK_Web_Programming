Cascading Style Sheets (CSS) adalah bahasa yang digunakan memberikan style pada halaman web yang ditulis dengan kode HTML.

Penggunaan CSS dapat dilakukan dengan beberapa cara yaitu :

  • Inline CSS

    Cara ini dituliskan langsung pada sebuah tag HTML dengan memberikan property dan value CSS di dalam atribut style pada tag HTML tersebut.

  • Internal CSS

    Penulisan CSS dengan cara ini dilakukan di dalam tag <head>, bukan di dalam tag <body>

  • Eksternal CSS

    External CSS adalah cara untuk menyimpan CSS pada sebuah file, kemudian file tersebut dapat digunakan oleh banyak halaman web.
